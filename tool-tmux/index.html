Tmux

Tmux是一种终端多路复用器:它允许在一个屏幕上创建、访问和控制多个终端。tmux可以从屏幕上分离并在后台继续运行，然后重新附加

详细教程可以看Tmux使用手册, 这里只记录我一些配置以备忘

安装

Linux: sudo apt install tmux

配置

tmux的配置文件是 ~/.tmux.conf，这个文件可能不存在，你可以自己新建。
