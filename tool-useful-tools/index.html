实用工具/脚本分享

发表于2020-07-15|更新于2021-08-05|Tool
|字数总计:926|阅读时长:3分钟|阅读量:

自用工具，想到就写进去, 里面的链接都是官网，请放心下载。

Windows版本: Windows 11 教育版 21H2

现用工具

utools: 类似wox的生产力工具合集，强推

Snipaste: 截图/贴图工具

Clash: 代理分流工具

Ditto: 剪贴板增强工具, 可以保存复制内容

Everything: 文件搜索神器

FileMenuTools: 右键增强工具

Folder Move: C盘空间不够用，又没钱买硬盘的方案, 原理是创建符号链接

PicGo: 图片上传+管理工具

Qtscrpy: 基于上面 scrcpy 工具开发的qt界面, 用这个就行了

Typora: 可视化编写md工具, 可搭配PicGo实现图片自动上传

RSS Guard: 开源的免费rss订阅器 + Feed43就可以订阅几乎所有了, 油猴脚本 RSS+可以用来显示页面的 RSS

Groupy: 将多个应用程序组织到Windows桌面上的分组选项卡中

wiztree: SpaceSniffer已停止更新。换成wiztree。用来搭配Folder Move

Tampermonkey/油猴: 一款免费的浏览器扩展和最为流行的用户脚本管理器

Visual Studio Code: 轻量级代码/文本编辑器

  • Markdown All in One : markdown 插件
  • Markdown Preview Enhanced: 预览 markdown
  • markdownlint: Markdown 语法检查插件
  • Material Icon Theme: Material 图标
  • Material Theme: Material 主题
  • PicGo: PicGo 插件
  • Sublime Text Keymap and Settings Importer: 可以在 vscode 里面使用 Sublime 的快捷键
  • TabNine: 超级好用的代码自动补全插件
  • Code Spell Checker: 代码拼写检查工具
  • Code Runner: 配置好了以后可以直接运行代码文件
  • C/C++: C/C++ 插件
  • Python: Python 插件
  • Insert DateTime: 用快捷键插入形如2020-10-21 09:32:54的时间字符串, 也可以修改模板使其更加符合你的使用规范

Microsoft To Do: 微软开发的任务清单, 支持安卓/PC/iOS 同步, 突然出现在我电脑的应用里, 觉得不错就拿来用了

Windows Terminal: 微软自己开发的终端窗口美化, 类似 cmder

WindowTop: Windows 窗口调整, 比如透明度, 置顶, 微缩等

微信/QQ/Tim 防撤回 + 微信多开: 如名, github 一个开源工具

tinyImage: 一款调用 tinypng.com 接口压缩图片的图片压缩工具

Bookxnote: 一款轻便的PDF管理与阅读器

OpenArk: Windows 热键冲突检查

Filelocator Pro: 和Everything差不多，但是其强大之处在于可以搜索文件内容

JFLAP: 画自动机的小工具
Altdrag: 按下alt键, 轻松拖动窗口。终于不用去找边角出现小箭头了！

ContextMenuManager: 鼠标右键菜单管理, 删掉一些没用的项目，让右键菜单更整洁

火绒: 安全软件, 个人免费

曾用工具

7+ Taskbar Tweaker: 控制任务栏

TranslucentTB: 一个轻量级的工具控制Windows任务栏，例如使任务栏透明【Store商店可下载】

SmartTaskbar: 可以控制任务栏，使得任务栏在最大化情况下隐藏任务栏【Store商店可下载】

TaskBar Hider: 直接隐藏状态栏, 但是UI界面设置快捷键无效，直接TaskbarHider.exe -hide有效, 搭配utools的快捷命令并设置鼠标中间快捷启用也挺有用的。合理搭配Windows快捷键

BitDock: 能像苹果一样弄一个底边栏奇怪的bug, 开启后没效果, 不用了。
Mouse without borders:微软的工具，它允许你用一个鼠标和键盘控制多达四台电脑。这意味着使用鼠标无边界，您可以复制文本或拖放文件在计算机之间。

SpaceSniffer: 磁盘文件探查工具

TeamViewer: 远程桌面控制

ToDesk: TeamViewer 莫名其妙卡上天, 换这个了（内网远控，不用了）

scrcpy: 电脑通过usb控制手机
